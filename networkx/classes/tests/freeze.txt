frozen graph
------------
>>> import networkx as nx
>>> G=nx.Graph([(1,2)])
>>> nx.is_frozen(G)
False
>>> G=nx.freeze(G)
>>> nx.is_frozen(G)
True
>>> try:
...    G.add_node(1)
... except nx.NetworkXError as e:
...    print(str(e))
Frozen graph can't be modified

>>> try:
...    G.add_edge(1,2)
... except nx.NetworkXError as e:
...    print(str(e))
Frozen graph can't be modified
